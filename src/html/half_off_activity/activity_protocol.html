活动协议

购车须知

1. 用户须确认自身具备相应的民事权利能力、民事行为能力及车辆购买资格，该等购车资格包含但不限于具有购买相应车辆所需要的机动车驾驶资格证及符合车辆上牌地的相应限购政策或其他政策（如有）；

2. 用户在汽车街平台成功支付订金后获得购车凭证，凭购车凭证在线下4S店可享受购买特卖车资格，凭购车凭证可抵用购车款1000元。

3. 用户在完成车款的选择并成功在线支付购车订金后，系统将向用户发送购车串码以确认下单成功，该订金为车辆出售方授权汽车街代车辆出售方收取；

4. 成功支付订金后，若48小时内未到店提车，汽车街客服将尽快致电用户与用户进一步确认购车意向，包括：购买车型信息、用户信息、需支付尾款信息、订单失效日期、提车地点等内容；若超过72小时用户未到线下提车，汽车街会取消订单并将定金在30日内退回给用户；

5. 若用户在下单支付成功后72小时之内申请退款，汽车街会在30日内将订金退回给用户，退款电话400-821-8889 。

3. 下单成功仅代表用户具备享有优惠购车的初步资格，并不作为最终购得车辆的凭证，下单后无法购买车辆的，请直接与出售方联系，汽车街仅作为交易提供平台及根据授权向用户收取订金，不对用户与出售方的行为承担任何责任；

5. 在用户到店提车时，车辆出售方将依据短信购车验证码确认用户的身份，为保证用户的权益，请用户务必妥善保存；
